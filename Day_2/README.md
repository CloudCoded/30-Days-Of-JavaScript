## ESCAPE SEQUENCES
<p> In JS, \ followed by some characters is an escape sequence. </p>
 
     for example:

     \n: new line
     \t: Tab, means 8 spaces
     \\: Back slash
     \': Single quote (')
     \": Double quote (")
